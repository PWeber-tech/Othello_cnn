# CSCI 447 Othello interactive model demo

This file will give you a guide to settup an enviornment to use this interactive demo to play othello/reversi with my generated models. This project was made for CSCI 447 Machine Learning

## how to settup

Settup is fairly simple download all the included files. 


### You will also need to have pytorch and pygame installed. 

The game is inside play_othello.py. Run this program to play with the models. 
All other files are imports do not try to run these.

All models created are inside checkpoints.
